# TaDa - backend
## Breakable toy

TaDa is a simple to-dos app. The backend was built with Java Spring

<img src="./spring-logo.png"  width="60" height="60">

To run the server do:

```
mvn spring-boot:run
```

To run the tests do:

```
mvn test
```
